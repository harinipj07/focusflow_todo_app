# FocusFlow To-Do Planner

This is a single HTML, CSS, JavaScript project.

## How to run

1. Extract the ZIP file.
2. Open the folder `focusflow_todo_app`.
3. Double-click `index.html`.

OR use VS Code:

1. Open the folder in VS Code.
2. Install the Live Server extension.
3. Right-click `index.html`.
4. Click "Open with Live Server".

## Features

- Login page
- Responsive landing/home page
- Smooth scroll navbar
- Light and dark theme toggle
- Daily, weekly, monthly planner buttons
- Add tasks with date and reminder time
- Calendar date click opens date-wise tasks
- Status indicator: Undone, In Progress, Completed
- Task reminders section
- Important notes section
- Daily completion progress bar like 10/15
- Local storage support
- Mobile, tablet, and desktop responsive
